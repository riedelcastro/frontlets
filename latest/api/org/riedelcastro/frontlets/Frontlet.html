Frontlet

class Frontlet extends AnyRef

A Frontlet provides typed access to an underlying Map data-structure. This map can come from various sources, such as JSON objects or MongoDB objects.

Frontlets are designed to make access to (often persistent) data safe, easy and generic. This comes at the price of efficiency, so clients should not access Frontlets in inner-loops of computations. In practice this may require a further conversion step from Frontlets into a more efficient representation.
